## 2026-05-06

**進度**：這兩天從變體中選題，目前傾向選擇 Dynamic 2D Rectangle BP 作為主題 Deep Dive 下去，詳細的比較內容都更新在 candidate-topics.md。接著我個人想到以前 Agent 使用 RL 學習走路的影片，是有趣的視覺化 Project，或許可以想想怎麼融入這個 Idea 進入我研究的視覺化。

**有去視覺化的 Idea**：
1. 俄羅斯方塊進化論 (Tetris-Style Survival)
與其讓 AI 靜態地填滿箱子，不如仿照影片中 Agent 跑障礙賽的感覺，讓箱子變成一個不斷下降或移動的空間。

視覺化概念：不同形狀的物件（3D 幾何體）像瀑布一樣落下。RL Agent 是一個「分揀員」或「機械手臂」，必須在物件落地前決定它的放置座標與旋轉角度。

RL 元素：

獎勵：成功消除一層（空間利用率）獲得高分；物品堆太高或掉出箱外則受到「痛覺懲罰」（畫面震動或 Agent 表現沮喪）。

視覺亮點：隨著訓練次數增加，你會看到 AI 從一開始亂丟、導致箱子瞬間爆滿，進化到像「俄羅斯方塊大神」一樣神速且精確地填補縫隙。
