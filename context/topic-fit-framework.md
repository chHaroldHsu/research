# 論文選題的契合度判斷框架

當思考一個候選題目時，可用以下三維度檢視：

## 維度 A：與楊老師方法論的契合度
- [ ] 是否包含可視化／視覺化呈現？
- [ ] 是否需要演算法設計或改良？
- [ ] 是否處理多媒體資料（影像、軌跡、空間資料）？
- [ ] 是否能做量化評估？

## 維度 B：與智慧製造系所的契合度
- [ ] 能否歸入「工業數據科學 / 智慧製造與營運 / 機器人工程與自動化技術」其中一類？
- [ ] 是否能對應到製造／物流／營運場域的實務問題？
- [ ] 即使取材自其他場域（智慧城市、交通），是否能做學術遷移論述？

## 維度 C：碩論可行性
- [ ] 資料是否取得得到（開放資料、模擬、自行蒐集）？
- [ ] 在 1～1.5 年內能否完成「baseline + 改良 + 評估」？
- [ ] 是否有清楚的 gap，能說明「別人還沒做好什麼」？
